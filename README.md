# python-web-bottle-params-hello-world

## Description
Simple web app that takes params
for a bottle project.

## Tech stack
- python
- bottle

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`
- http://localhost
- http://localhost/<name>
  - Where <name> is a string

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
